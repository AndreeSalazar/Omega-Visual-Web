# Videos Folder

Place your hero video here:

- **File name:** `hero-demo.mp4`
- **Recommended resolution:** 1920x1080 (Full HD)
- **Format:** MP4 (H.264 codec recommended for best compatibility)
- **Duration:** 30-60 seconds (looping)
- **File size:** Try to keep under 10MB for faster loading

The video will automatically play when the user clicks the play button on the hero image.

## Video Specifications

- **Codec:** H.264
- **Container:** MP4
- **Resolution:** 1920x1080 (or 16:9 aspect ratio)
- **Frame rate:** 30fps or 60fps
- **Background:** Black background recommended
- **Content:** Should showcase the Omega-Visual node canvas in action

## Tips

- Compress the video to reduce file size while maintaining quality
- Use tools like HandBrake or FFmpeg for optimization
- Ensure the video loops smoothly (last frame should transition to first frame)
